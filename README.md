# 📝 Task Scheduling App

A Flutter app to manage tasks using speech-to-text and AI (Gemini) integration.

## ✨ Features

- 🎙️ Voice-controlled task creation
- 🧠 Google Gemini AI integration
- 📅 Smart date and time recognition
- 🗂️ BLoC state management
